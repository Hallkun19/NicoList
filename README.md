# NicoList - 動画リスト拡張

ニコニコ動画・YouTube に対応した、ブラウザ上で動画をリスト管理できる Chrome 拡張機能です。
> [!NOTE]
> 友人(https://www.nicovideo.jp/user/132889643)がAIで作成した拡張機能の開発を引き継ぎました。
> そのため大半がAIで生成されているコードになっています、不備などがあったら随時修正していきます

## 機能

- **リスト管理** — 複数のリストを作成し、動画を自由に追加・削除・並び替え
- **クイック追加** — 動画ページのボタンからワンクリック／ダブルクリックでリストに即追加
- **連続再生・シャッフル再生** — リスト内の動画を順番またはランダムに連続再生
- **大画面ビュー** — 新しいタブで全画面リスト表示（グリッド／リスト切替・ソート対応）
- **リスト共有** — リストをコードにエンコードして他のユーザーと共有
- **動画情報の自動取得** — 再生数・いいね数・マイリスト数・投稿日・サムネイルなどを API から取得
- **ローカルキャッシュ** — 取得した動画情報を 24 時間キャッシュし、API への負荷を軽減
- **情報更新** — ボタン一つでリスト内の動画情報を一括再取得
- **インポート / エクスポート** — JSON 形式でリストデータを保存・復元
- **ダークモード対応** — 各サイトのテーマに合わせて自動切替

## インストール

1. [Releases](../../releases) から最新の ZIP をダウンロード
2. ZIP を展開
3. `chrome://extensions` を開く
4. 右上の「デベロッパー モード」を有効化
5. 「パッケージ化されていない拡張機能を読み込む」→ 展開したフォルダを選択

## 使い方

### 動画をリストに追加

1. ニコニコ動画または YouTube の動画ページを開く
2. プレイヤー付近に表示される **NicoList ボタン**をクリック
3. 追加先のリストを選択

> **ヒント**: 設定でデフォルトリストを指定すると、ダブルクリック（または設定でクリック）で即追加できます。

### リストの共有

1. ポップアップまたは大画面ビューでリストを開く
2. 「共有」ボタンからコードを生成
3. コードを相手に送る → 相手はコード付き URL を開くだけで閲覧可能

## 技術仕様

| 項目 | 内容 |
|------|------|
| マニフェスト | Manifest V3 |
| データ保存 | IndexedDB (`NicoListDB`) |
| キャッシュ | `chrome.storage.local`（24h TTL） |
| 対応サイト | ニコニコ動画 / YouTube |
